Nom et description du projet:
Jeu de Mémoire Avancé - Un jeu de mémoire interactif développé en pur JavaScript avec système de scores, chronomètre et cartes spéciales.

Technologies utilisées:
HTML5, CSS3, JavaScript ES6+

LocalStorage API pour la sauvegarde des scores

Git/GitHub pour le versionning

Fonctionnalités principales:
3 niveaux de difficulté (8, 12, 16 cartes)

Système de scoring avec cartes bonus/malus

Chronomètre avec bonus de rapidité

Sauvegarde des meilleurs scores


Lien GitHub Pages
[https://aymenrhf.github.io/Rhimi_Aymen_jeu_memoire/]

Apprentissages techniques :
